The zipped file in this folder was downloaded on June 16th, 2025. It contains all the data files and documentation that come with a 
download of the National Provider Identifier monthly replacement file. Some of the files are very large. Unzip this file folder on your 
local machine within the \chrr_measure_calcs\raw_data\NPI folder. 

The npidata_pfile_20050523-20250608.csv is 10GB when unzipped and is used to calculate v062 (Mental Health Providers) and v131 (Other 
Primary Care Providers).
